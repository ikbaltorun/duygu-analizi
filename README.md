# duygu-analizi
Python, OpenCV ve DeepFace kütüphaneleri kullanılarak geliştirilmiş gerçek zamanlı duygu analizi projesi. Bilgisayar kamerası üzerinden yüz tespiti yapar ve hareketli ortalama (son 10 kare) yöntemini kullanarak kişinin anlık mutluluk/gülümseme oranını pürüzsüz bir şekilde ekrana yansıtır.
